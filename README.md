🚦 AIoT Traffic Predictor
Prédiction intelligente du trafic routier en temps réel
📌 Description

AIoT Traffic Predictor est une application intelligente qui utilise les concepts de l’Intelligence Artificielle (IA) et de l’Internet des Objets (IoT) pour analyser et prédire l’état du trafic routier en temps réel.

Ce système permet d’anticiper les conditions de circulation (fluide ou congestionnée) à partir de données simples, simulant un environnement de Smart City.

🎯 Objectifs
Prédire l’état du trafic en temps réel
Améliorer la gestion de la mobilité urbaine
Réduire les embouteillages
Fournir un outil simple d’aide à la décision
⚙️ Fonctionnalités
🚗 Saisie du volume de trafic
🤖 Prédiction automatique via un modèle IA
🟢 Affichage “Traffic fluide” (circulation normale)
🔴 Détection des embouteillages
🎯 Interface moderne et intuitive
⚡ Résultat instantané
🧠 Logique du système

L’application analyse une valeur d’entrée (densité du trafic) et applique une logique de classification :

Faible trafic → Circulation fluide
Trafic élevé → Embouteillage détecté

Notre modèle sera amélioré avec :

Machine Learning (Random Forest, LSTM…)
Données IoT (capteurs, GPS, caméras)
🖥️ Interface utilisateur

L’application propose une interface simple avec :

Champ de saisie du trafic
Bouton de prédiction
Résultat visuel avec indicateurs
📸 Aperçu
🟢 Cas : trafic fluide
Entrée faible (ex: 12)
Résultat : circulation normale
🔴 Cas : embouteillage
Entrée élevée (ex: 1200)
Résultat : trafic dense
