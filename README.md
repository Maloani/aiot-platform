# AIoT Platform 🚀

Plateforme intelligente combinant IoT + IA pour la prédiction du trafic.

## Fonctionnalités
- Simulation de capteurs IoT
- Modèle Machine Learning (Random Forest)
- API Flask
- Dashboard web
- MQTT + Streaming
